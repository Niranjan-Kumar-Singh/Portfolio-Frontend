### Run Locally

1. **Clone the Repository**:

```bash
git clone https://github.com/Niranjan-singh-code/portfolio.git
```

2. **Navigate to the Directory**:

```bash
cd portfolio
```

3. **Install dependencies**:

```bash
npm install
```

4. **Start development server**:

```bash
npm run dev
```
